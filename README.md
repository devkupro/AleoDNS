### Discord Members
- vanphappi314 (Leader)

1. Clone the source code from the repository:

```bash
git clone https://github.com/devkupro/AleoDNS.git
```
2. Use Makefile to run

- run server by CLI
```bash
make backend
```

- run UI by CLI
```bash
make frontend
```
