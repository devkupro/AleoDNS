## Getting started
1. Clone the source code from the repository:

```bash
git clone git@gitlab.rinznetwork.com:factory/backend/go-api.git
```
2. Use Makefile to run

- run server by CLI
```bash
make backend
```

- run UI by CLI
```bash
make frontend
```
